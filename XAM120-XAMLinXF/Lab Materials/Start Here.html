XAML in Xamarin.Forms

Before you begin

This is a Xamarin.Forms class. To complete the coding exercises, you will need a Xamarin development environment with either a simulator/emulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This session introduces the XAML parser built into Xamarin.Forms and shows how to create your user interfaces declaratively and then interact with the screens using code behind.

Objectives

  1. XAML syntax
  2. Adding Behavior to XAML-based UI
  3. Advanced XAML

Lecture

XAM130 - XAML in Xamarin.Forms.

Exercises

  1. Create a XAML-based version of Calculator
    Exercise
    This exercise takes an existing Xamarin.Forms application that is created completely in code and converts it to use XAML to create the UI.
  2. Adding Behavior to XAML Calculator
    Exercise
    In this exercise we will connect the XAML-based UI to the code in the Calculator code behind.
  3. Cleanup the XAML code and tailor the UI to the platform
    Exercise
    This final exercise shares some of the common values in the XAML and ensures that it looks good on each platform by tweaking the UI with platform-specific values.
